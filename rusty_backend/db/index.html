§Модуль работы с базой данных

Этот модуль предоставляет абстракцию для работы с базой данных PostgreSQL. Использует SQLx для асинхронных запросов и включает структурированное логирование.

§Основные возможности

  • Управление пользователями: создание, чтение, обновление, удаление
  • Управление серверами: создание серверов, управление участниками
  • Текстовые каналы: создание каналов, управление сообщениями
  • Безопасность: хеширование паролей с bcrypt

§Примеры использования

use rusty_backend::db::Db;

// Создание подключения
let db = Db::new("postgres://user:pass@localhost/db").await?;

// Создание пользователя
db.create_user("John", "john@example.com", None, "hashed_password").await?;

// Получение пользователя
let user = db.get_user_by_id(1).await?;

§Логирование

Все операции с базой данных логируются с помощью tracing:

  • Входные параметры (кроме паролей)
  • Результаты операций
  • Ошибки с детальной информацией
