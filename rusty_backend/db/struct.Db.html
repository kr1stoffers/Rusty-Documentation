Struct Db

pub struct Db {

Expand description

Представляет подключение к базе данных PostgreSQL.

Эта структура инкапсулирует пул соединений к базе данных и предоставляет методы для выполнения SQL запросов.

§Примеры

use rusty_backend::db::Db;

let db = Db::new("postgres://user:pass@localhost/db").await?;

pub async fn new(database_url: &str) -> Result<Self, Error>

Создает новое подключение к базе данных.

Инициализирует пул соединений к PostgreSQL с указанным URL.

§Аргументы
  • database_url - URL подключения к PostgreSQL в формате postgres://username:password@host:port/database
§Примеры
use rusty_backend::db::Db;

let db = Db::new("postgres://admin:root@localhost:5432/postgres").await?;
§Ошибки

Возвращает ошибку если:

  • Неверный URL подключения
  • Не удалось подключиться к базе данных
  • Неверные учетные данные

pub async fn create_user( &self, name: &str, email: &str, phone: Option<&str>, password: &str, ) -> Result<(), Error>

Создает нового пользователя в базе данных.

Добавляет нового пользователя с захешированным паролем. Email должен быть уникальным в системе.

§Аргументы
  • name - Имя пользователя
  • email - Email пользователя (должен быть уникальным)
  • phone - Номер телефона (опционально)
  • password - Захешированный пароль
§Примеры
use bcrypt;

let hashed_password = bcrypt::hash("password", 4)?;
db.create_user("John", "john@example.com", None, &hashed_password).await?;
§Ошибки

Возвращает ошибку если:

  • Email уже существует
  • Неверные данные (слишком длинные строки)
  • Проблемы с подключением к БД
