§Миграции базы данных

Этот модуль содержит функции для создания и управления схемой базы данных. Использует SQLx для выполнения SQL миграций.

§Структура базы данных

  • user - Пользователи системы
  • server - Серверы с участниками
  • server_member - Связь пользователей с серверами
  • text_channel - Текстовые каналы на серверах
  • message - Сообщения в каналах

§Примеры использования

use rusty_backend::migration::create_tables;
use sqlx::PgPool;

let pool = PgPool::connect("postgres://...").await?;
create_tables(&pool).await?;
