§HTTP обработчики

Этот модуль содержит все HTTP обработчики для API endpoints. Каждый обработчик включает структурированное логирование и обработку ошибок.

§Группы обработчиков

  • Управление пользователями: регистрация, авторизация, CRUD операции
  • Управление серверами: создание серверов, управление участниками
  • Текстовые каналы: создание каналов, отправка сообщений
  • Голосовые каналы: WebRTC интеграция

§Логирование

Все обработчики используют tracing для структурированного логирования:

  • Каждый запрос создает span с контекстными полями
  • Логируются входные параметры и результаты
  • Ошибки логируются с детальной информацией
