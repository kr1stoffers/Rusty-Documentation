Создает новое сообщение в текстовом канале.

Добавляет сообщение от указанного пользователя в канал.

§Endpoint

POST /server/text_channel/{text_channel_id}/user/{author_id}/message

§Параметры

  • text_channel_id - Уникальный идентификатор текстового канала
  • author_id - Уникальный идентификатор автора сообщения

§Примеры

curl -X POST http://localhost:8080/server/text_channel/1/user/123/message \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Привет, всем!"
  }'

§Ответы

  • 201 Created - Сообщение успешно создано
  • 500 Internal Server Error - Ошибка сервера
