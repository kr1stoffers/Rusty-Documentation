Добавляет пользователя на сервер.

Делает пользователя участником указанного сервера.

§Endpoint

POST /server/{server_id}/member

§Параметры

  • server_id - Уникальный идентификатор сервера

§Примеры

curl -X POST http://localhost:8080/server/1/member \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": 123
  }'

§Ответы

  • 201 Created - Пользователь успешно добавлен на сервер
  • 500 Internal Server Error - Ошибка сервера
