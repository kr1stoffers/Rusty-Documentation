§Модели данных

Этот модуль содержит определения структур данных, используемых в приложении. Все структуры поддерживают сериализацию/десериализацию с помощью Serde.

§Основные модели

  • User - Пользователь системы
  • LoginUser - Данные для авторизации
  • Server - Сервер с участниками
  • TextChannel - Текстовый канал
  • Message - Сообщение в канале

§Примеры использования

use rusty_backend::model::{User, Server, Message};

// Создание пользователя
let user = User {
    id: 1,
    name: "John Doe".to_string(),
    email: "john@example.com".to_string(),
    phone: None,
    avatar: None,
};

// Сериализация в JSON
let json = serde_json::to_string(&user)?;
