§Rusty Library - Общая библиотека

Общая библиотека для backend и SFU серверов. Содержит общие структуры данных, типы и утилиты.

§Основные модули

  • event - События и типы для WebRTC
  • Общие структуры данных
  • Утилиты для работы с WebRTC

§Примеры использования

use rusty_lib::event::{WebRTCEvent, EventType, Event};
use webrtc::peer_connection::sdp::session_description::RTCSessionDescription;

// Создание события Join
let sdp = RTCSessionDescription::new(webrtc::peer_connection::sdp::session_description::RTCSdpType::Offer, "sdp_data".to_string());
let join_event = Event::new_join(123, 456, sdp);

// Создание события подключения
let event = WebRTCEvent {
    event_type: EventType::ConnectionEstablished,
    user_id: 123,
    channel_id: 456,
    data: Some("connection_data".to_string()),
};
