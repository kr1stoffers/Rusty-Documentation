Rust Documentation

Добро пожаловать в документацию проекта

Этот проект содержит несколько Rust крейтов, каждый из которых выполняет свою роль в системе.

Доступные крейты

  • rusty_backend
    Веб-сервер на Actix Web для управления пользователями, серверами и каналами
  • rusty_lib
    Общая библиотека с событиями и типами данных
  • rusty_sfu
    SFU (Selective Forwarding Unit) для WebRTC голосового общения

Быстрый старт

Для начала работы с проектом рекомендуется ознакомиться с документацией rusty_backend, который является основным компонентом системы.
